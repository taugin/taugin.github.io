正则表达式测试工具

在线正则表达式测试工具，实时测试正则表达式并高亮匹配结果，支持多种匹配模式。

等待匹配...

工具说明

功能介绍

  • 实时匹配: 输入正则表达式和文本后实时显示匹配结果
  • 高亮显示: 匹配的内容将以高亮形式显示在结果区域
  • 匹配信息: 显示匹配的数量、位置等详细信息
  • 替换功能: 支持使用正则表达式进行文本替换
  • 多种模式: 支持全局匹配(g)、忽略大小写(i)、多行模式(m)等标志

使用方法

  1. 在"正则表达式"输入框中输入要测试的正则表达式
  2. 选择需要的匹配标志(可选)
  3. 在"测试文本"输入框中输入要测试的文本
  4. 查看高亮显示的匹配结果和匹配信息
  5. 如需替换，可在"替换文本"中输入替换内容并点击"执行替换"

常见正则表达式示例

  • 匹配数字: \d+
  • 匹配邮箱: \b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b
  • 匹配URL: https?:\/\/(www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b([-a-zA-Z0-9()@:%_\+.~#?&//=]*)
  • 匹配手机号: 1[3-9]\d{9}
