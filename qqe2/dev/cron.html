Cron表达式生成

在线Cron表达式生成工具，可视化配置定时任务执行时间。

配置选项

工具说明

Cron表达式格式

Cron表达式由五个或六个字段组成，按顺序分别是：

  1. 分钟 - 0到59
  2. 小时 - 0到23
  3. 日 - 1到31
  4. 月 - 1到12
  5. 周 - 0到7 (0和7都表示周日)

特殊字符说明

  • * - 匹配该字段的所有值
  • ? - 不指定值，用于日和周字段
  • - - 表示范围，如1-5
  • , - 表示列举，如1,3,5
  • / - 表示增量，如0/5表示从0开始每5个单位

使用示例

  • 0 0 * * * - 每天0点执行
  • 0 12 * * * - 每天12点执行
  • 0 0 * * 1 - 每周一0点执行
  • 0 0 1 * * - 每月1号0点执行
