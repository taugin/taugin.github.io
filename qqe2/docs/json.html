JSON教程

JSON基础知识、语法和使用教程，帮助开发者更好地理解和使用JSON格式。

什么是JSON？

JSON (JavaScript Object Notation) 是一种轻量级的数据交换格式。它基于JavaScript的一个子集，但它是独立于语言的文本格式，被广泛应用于Web开发中用于数据传输和存储。

JSON语法规则

  • 数据以键值对的形式存在
  • 数据由逗号分隔
  • 大括号保存对象
  • 方括号保存数组

JSON数据类型

  • 字符串 - 必须用双引号包围，如 "name"
  • 数字 - 整数或浮点数，如 42 或 3.14
  • 布尔值 - true 或 false
  • null - 空值
  • 对象 - 无序的键值对集合，用大括号 {} 包围
  • 数组 - 有序的值列表，用方括号 [] 包围

JSON示例

{
  "name": "张三",
  "age": 30,
  "isStudent": false,
  "address": {
    "street": "中山路123号",
    "city": "北京"
  },
  "hobbies": ["读书", "游泳", "编程"],
  "spouse": null
}

JSON与XML对比

特性 JSON XML
数据体积 更小 较大
解析难度 简单 复杂
可读性 较好 好
数据类型 丰富 有限

JSON使用场景

  • Web应用程序之间的数据传输
  • API接口返回数据格式
  • 配置文件存储
  • Ajax请求的数据格式
  • 前后端数据交互
