# 可复用的工具页面模板 Prompt

这个 Prompt 模板可用于快速生成新的工具功能页面。只需替换相应的内容，即可快速创建一个新的工具页面。

## Prompt 模板

```
你是一个专业的前端开发工程师，请帮我创建一个"[工具名称]"工具页面，用于我的 GitHub Pages 静态网站项目。

要求：
1. 使用 HTML5、TailwindCSS 和原生 JavaScript 实现
2. 页面包含以下功能区域：
   - 顶部导航栏（返回首页链接）
   - 工具标题和简要说明
   - 输入区域（文本框或表单元素）
   - 功能按钮（执行操作、清空、复制等）
   - 实时结果显示区域
   - 使用说明区域
   - 页脚（版权信息）

具体功能要求：
- 输入框支持多行文本输入，使用等宽字体
- 至少包含一个主要功能按钮"[主要功能按钮文本]"
- 提供清空输入和复制结果功能
- 实时显示操作结果
- 出错时显示友好的错误信息
- 响应式设计，适配移动端和桌面端
- 使用 TailwindCSS 实现现代化界面设计

页面结构要求：
- 页面标题：<title>[工具名称] - 在线工具箱</title>
- 页面描述：<meta name="description" content="[工具描述，包括主要功能和特点]">
- 包含基本的 SEO meta 标签和 Open Graph 标签
- 使用语义化 HTML 结构
- JavaScript 代码放在页面底部，在 DOM 加载完成后执行

设计规范：
- 主色调使用 indigo（Tailwind 的 indigo 颜色系列）
- 成功状态使用 green，错误状态使用 red
- 按钮样式：主要操作使用 bg-indigo-600，成功操作使用 bg-green-600，危险操作使用 bg-red-600
- 输入框和结果区域使用等宽字体（Monaco, Menlo, Ubuntu Mono, monospace）
- 卡片阴影使用 Tailwind 的 shadow-sm 或 shadow-md
- 圆角使用 rounded-md

请输出完整的 HTML 文件代码，包含：
1. HTML 结构
2. CSS 样式（内联在<style>标签中）
3. JavaScript 功能实现
4. 使用说明文档

工具信息：
工具名称：[在此处填写工具名称]
工具描述：[在此处填写工具的详细描述]
主要功能：[在此处描述工具的主要功能]
输入类型：[文本/数字/代码等]
输出类型：[文本/数字/代码等]
功能按钮文本：[在此处填写主要功能按钮的文本]
```

## 使用说明

1. 复制上述 Prompt 模板
2. 替换方括号中的占位内容：
   - `[工具名称]`：工具的名称
   - `[工具描述]`：工具的详细描述
   - `[主要功能]`：工具的主要功能说明
   - `[输入类型]`：工具接受的输入类型
   - `[输出类型]`：工具产生的输出类型
   - `[功能按钮文本]`：主要功能按钮上显示的文本

3. 将完整的 Prompt 发送给 AI 助手生成工具页面代码
