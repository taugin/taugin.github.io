# GitHub 静态网站测试检查表

本文档提供了全面的测试检查清单，确保静态网站在发布前达到高质量标准。该检查表涵盖了功能、兼容性、性能、安全、SEO 和 UI 等各个方面。

## 1. 功能测试

### 1.1 核心功能测试
- [ ] 首页正常加载，所有工具卡片显示正确
- [ ] 搜索功能正常工作，能够正确筛选工具
- [ ] 所有工具页面均可正常访问
- [ ] 工具功能按预期工作（JSON 校验、Base64 编解码等）
- [ ] 复制到剪贴板功能正常
- [ ] 清空输入功能正常
- [ ] 表单输入和输出区域正常工作
- [ ] 错误处理机制有效（如 JSON 格式错误提示）

### 1.2 导航测试
- [ ] 从首页可以访问所有工具页面
- [ ] 从工具页面可以返回首页
- [ ] 页面间跳转流畅，无死链接
- [ ] 面包屑导航（如果有）正常工作

### 1.3 用户交互测试
- [ ] 按钮点击响应迅速
- [ ] 输入框可以正常输入和编辑
- [ ] 文本选择和复制功能正常
- [ ] 键盘快捷键（如果有）正常工作
- [ ] 页面滚动流畅，无卡顿

## 2. 兼容性测试

### 2.1 桌面浏览器兼容性
- [ ] Chrome 最新版功能正常
- [ ] Firefox 最新版功能正常
- [ ] Safari 最新版功能正常
- [ ] Edge 最新版功能正常
- [ ] Opera 最新版功能正常

### 2.2 移动设备兼容性
- [ ] iOS Safari 功能正常
- [ ] Android Chrome 功能正常
- [ ] 移动端页面布局正确
- [ ] 触摸操作响应灵敏
- [ ] 移动端输入法兼容性良好

### 2.3 不同屏幕尺寸适配
- [ ] 320px 宽度（iPhone SE）显示正常
- [ ] 375px 宽度（iPhone 14）显示正常
- [ ] 414px 宽度（iPhone Plus）显示正常
- [ ] 768px 宽度（iPad）显示正常
- [ ] 1024px 宽度（iPad Pro）显示正常
- [ ] 1280px 及以上宽度显示正常

## 3. 性能测试

### 3.1 加载性能
- [ ] 首页加载时间不超过 3 秒
- [ ] 工具页面加载时间不超过 2 秒
- [ ] 首屏渲染时间合理
- [ ] 关键资源加载优先级正确

### 3.2 运行性能
- [ ] JavaScript 执行无明显延迟
- [ ] 页面交互响应时间小于 100ms
- [ ] 大数据处理时不阻塞 UI 线程
- [ ] 内存占用合理，无持续增长

### 3.3 资源优化
- [ ] CSS 和 JS 文件已压缩
- [ ] 图片已优化（格式、尺寸、压缩）
- [ ] 使用了适当的缓存策略
- [ ] 无未使用的资源加载

## 4. 安全检查

### 4.1 XSS 防护
- [ ] 用户输入内容正确转义显示
- [ ] 无执行用户输入的 JavaScript 代码
- [ ] HTML 内容插入使用安全方法
- [ ] URL 参数正确处理和验证

### 4.2 内容安全
- [ ] 无敏感信息泄露
- [ ] 错误信息不暴露系统细节
- [ ] 无硬编码的敏感信息（如密钥）
- [ ] 第三方资源加载安全可靠

### 4.3 依赖风险检查
- [ ] 使用的第三方库版本安全
- [ ] CDN 资源来自可信源
- [ ] 无已知漏洞的依赖项
- [ ] 定期更新第三方库

## 5. SEO 检查

### 5.1 基础 SEO
- [ ] 每个页面有唯一且相关的 title 标签
- [ ] 每个页面有准确的 description 标签
- [ ] 使用了适当的 header 标签结构（h1, h2, h3 等）
- [ ] 图片包含 alt 属性
- [ ] 页面有合理的内部链接结构

### 5.2 Open Graph 和社交媒体
- [ ] Open Graph 标签设置正确
- [ ] Twitter Card 标签设置正确
- [ ] 社交媒体分享预览显示正常
- [ ] 网站图标（favicon）设置正确

### 5.3 技术 SEO
- [ ] robots.txt 文件配置正确
- [ ] sitemap.xml 文件生成并可访问
- [ ] 页面响应码正确（200 OK）
- [ ] 无重复内容问题
- [ ] URL 结构清晰且一致

## 6. UI 检查

### 6.1 响应式设计
- [ ] 页面在各种屏幕尺寸下显示正常
- [ ] 文本大小在不同设备上可读
- [ ] 按钮和交互元素在触摸设备上易于操作
- [ ] 图片和媒体元素正确缩放

### 6.2 视觉一致性
- [ ] 颜色方案统一
- [ ] 字体使用一致
- [ ] 按钮和控件样式统一
- [ ] 页面间距和对齐合理

### 6.3 暗色模式（如果支持）
- [ ] 暗色模式切换功能正常
- [ ] 暗色模式下文本可读性良好
- [ ] 暗色模式下颜色对比度符合标准
- [ ] 暗色模式设置持久化保存

### 6.4 可访问性
- [ ] 页面可通过键盘完全操作
- [ ] 屏幕阅读器可正确解读内容
- [ ] 色彩对比度符合 WCAG 标准
- [ ] 表单元素有适当的标签
