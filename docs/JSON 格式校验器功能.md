# JSON 格式校验器工具文档

## 1. 工具概述

JSON 格式校验器是一个在线工具，用于校验、格式化和压缩 JSON 数据。该工具可以帮助开发者快速验证 JSON 语法的正确性，并提供格式化和压缩功能，使 JSON 数据更易读或更紧凑。

## 2. 功能特性

### 2.1 主要功能

1. **JSON 校验**
   - 实时检查 JSON 格式是否正确
   - 提供详细的错误信息和位置指示
   - 支持中文错误提示

2. **JSON 格式化**
   - 将压缩的 JSON 数据转换为易读的格式
   - 自动缩进和换行
   - 保持数据结构清晰

3. **JSON 压缩**
   - 移除所有不必要的空白字符
   - 减小 JSON 数据体积
   - 适用于生产环境传输

4. **结果复制**
   - 一键复制处理结果到剪贴板
   - 提高工作效率

5. **输入清空**
   - 快速清空输入内容
   - 重新开始新的校验任务

### 2.2 界面特性

1. **响应式设计**
   - 适配各种屏幕尺寸
   - 移动端友好

2. **可视化反馈**
   - 成功和错误状态的颜色区分
   - 实时状态指示器

3. **用户友好**
   - 直观的操作按钮布局
   - 清晰的使用说明

## 3. 使用说明

### 3.1 基本操作流程

1. 在输入框中粘贴或输入 JSON 文本
2. 点击"校验 JSON"按钮检查 JSON 格式是否正确
3. 查看结果区域的输出和状态提示
4. 根据需要使用格式化或压缩功能
5. 使用复制功能获取处理后的结果

### 3.2 详细操作指南

#### 校验 JSON
- 点击"校验 JSON"按钮验证输入的 JSON 格式
- 如果格式正确，会在结果区域显示格式化的 JSON
- 如果格式错误，会显示具体的错误信息和位置

#### 格式化 JSON
- 点击"格式化"按钮将 JSON 转换为易读格式
- 添加适当的缩进和换行
- 保持数据结构清晰可见

#### 压缩 JSON
- 点击"压缩"按钮移除所有空白字符
- 生成紧凑的 JSON 字符串
- 适用于网络传输和存储

#### 复制结果
- 点击"复制结果"按钮将输出内容复制到剪贴板
- 可以粘贴到其他应用程序中使用

#### 清空输入
- 点击"清空"按钮清除所有输入内容
- 重新开始新的校验任务

## 4. 技术实现

### 4.1 前端技术栈

- **HTML5**: 页面结构和语义化标记
- **Tailwind CSS**: 样式设计和响应式布局
- **原生 JavaScript**: 交互逻辑和数据处理

### 4.2 核心功能实现

#### JSON 校验
```javascript
try {
  const parsed = JSON.parse(jsonString);
  // 格式化并显示结果
} catch (error) {
  // 显示错误信息
}
```

#### JSON 格式化
```javascript
const parsed = JSON.parse(jsonString);
const formatted = JSON.stringify(parsed, null, 2);
```

#### JSON 压缩
```javascript
const parsed = JSON.parse(jsonString);
const compressed = JSON.stringify(parsed);
```

## 6. 错误处理

### 6.1 常见错误类型

1. **语法错误**
   - 缺少引号
   - 缺少逗号
   - 多余逗号
   - 括号不匹配

2. **数据类型错误**
   - 使用单引号代替双引号
   - 属性名未加引号
   - 末尾多余的逗号

### 6.2 错误提示

- 显示具体的错误信息
- 提供错误发生的行号
- 在结果区域高亮显示错误内容
